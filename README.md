## 프로젝트 소개

<p align="justify">
디지털 하나로 수강생, 희망생에게 정보를 제공하고 소통할 수 있는 장을 마련
</p>
<p align="justify">
Project Hosting : 디지털 하나로 1기 5조
</p>
<p align="justify">
개발 기간 : 2023.5~2023.6
</p>

## 구현 기능

### 메인 페이지

- 카카오 로그인
- 메인 캐러셀
- 원형 룰렛, 카카오 지도 API
- QNA, 1대 1 문의 챗봇

### 소개 페이지

- 스크롤 이벤트

### 지원 페이지

- 스크롤 이벤트

### 교육 페이지

- 월별 교육 캘린더
- 강사 소개

### 커뮤니티 페이지

- 글 작성 페이지
- slack과 연동된 공지사항 페이지
