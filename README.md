## 구현 기능

### 메인 페이지

- 카카오 로그인
- 메인 캐러셀
- 원형 룰렛, 카카오 API
- QnA, 슬랙과 연동된 1대 1 문의 챗봇

### 소개 페이지

- 스크롤 이벤트

### 지원 페이지

- 스크롤 이벤트

### 교육 페이지

- 월별 교육 캘린더
- 강사 소개

### 커뮤니티 페이지

- 글 작성 페이지
- slack과 연동된 공지사항 페이지
